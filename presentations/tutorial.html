Faza 1 — Import i orientacja

Otwórz Gephi Lite i zaimportuj plik

Gephi Lite działa w przeglądarce — nie wymaga instalacji. Otwórz adres lite.gephi.org w Chrome lub Firefox.

Jak wczytać plik

Na ekranie powitalnym kliknij „Open a local file" i wybierz graf_postaci.graphml z dysku.

📂 Ścieżka klikania:
Open a local file → wybierz graf_postaci.graphml → Open
💡 Gephi Lite obsługuje formaty GraphML, GEXF, JSON Graph. Twój plik jest w formacie .graphml — zostanie wczytany automatycznie bez konwersji.

Po załadowaniu na ekranie zobaczysz rozproszone węzły bez żadnego układu — to normalne. Sieć ma 47 węzłów i 103 krawędzie.

Faza 2 — Układ przestrzenny (Layout)

Zastosuj układ ForceAtlas 2

ForceAtlas 2 to algorytm siłowo-skierowany, który automatycznie układa węzły tak, że silnie połączone postacie lądują blisko siebie, a słabo połączone — daleko. To serce każdej wizualizacji w Gephi Lite.

🔧 Ścieżka klikania:
Lewy panel → Layout → ForceAtlas 2 → ▶ Run

Zalecane ustawienia dla tej sieci

Scaling100
Gravity1.0
Prevent Overlap✓ włączone
Edge Weight Influence1.0
LinLog modenie zmieniaj

Obserwuj, jak JACK i TYLER przyciągają się do centrum — to efekt ich wagi weight=40. Gdy animacja się ustabilizuje, kliknij ■ Stop.

💡 Jeśli sieć wygląda zbyt ściśnięta — zwiększ Scaling do 200. Jeśli węzły się nakładają mimo „Prevent Overlap" — uruchom ForceAtlas 2 ponownie przez kilka sekund po zatrzymaniu.

Alternatywny układ: Noverlap

Po zatrzymaniu ForceAtlas 2 możesz uruchomić Noverlap jako post-processing — rozsuwa on węzły bez zmiany ogólnej struktury. Polecane gdy węzły się nakładają.

✨ Opcjonalnie:
Layout → Noverlap → ▶ Run → ■ Stop po ~2 sek.
Faza 3 — Metryki sieci

Oblicz metryki: Degree, Modularity, Betweenness

Metryki są kluczowe — pozwalają zmierzyć znaczenie każdej postaci i znaleźć wspólnoty. Obliczone wartości staną się atrybutami węzłów, które za chwilę wykorzystamy w wyglądzie.

📐 Ścieżka klikania:
Lewy panel → Statistics

④ Betweenness Centrality

Kliknij Run przy Network Diameter / Betweenness Centrality. Mierzy, jak często dany węzeł leży „na drodze" między innymi. JACK i TYLER mają tutaj bardzo wysokie wartości — są mostami informacyjnymi fabuły.

💡 Po obliczeniu metryk wróć do zakładki Data i sprawdź tabelę Nodes — zobaczysz nowe kolumny: degree, modularity_class, betweenness centrality.

💡 Możesz też użyć Weighted Degree zamiast Degree — da to jeszcze wyraźniejszy efekt, bo JACK z Tylerem mają łącznie bardzo dużą wagę.

Koloruj węzły według Modularity Class

Każda wspólnota (klaster) dostanie swój kolor — od razu zobaczysz grupy postaci: Space Monkeys, trio główne, detektywi, epizodyczne.

🎨 Ścieżka klikania:
Appearance → Nodes → Color (ikona palety) → Partition

Z listy wybierz modularity_class. Gephi Lite automatycznie przypisze kolory. Kliknij Apply.

💡 Możesz ręcznie zmienić kolor każdej klasy klikając na kolorowy kwadrat przy każdej wartości modularity. Np. ustaw klasę JACK+TYLER na czerwony, Space Monkeys na niebieski.

Alternatywa: kolor według Betweenness Centrality

Wybierz Ranking zamiast Partition i atrybut betweenness centrality — zobaczysz gradientową skalę od zimnych (peryferyjnych) do gorących (centralnych) kolorów.

Faza 5 — Etykiety i czytelność

Włącz i dostosuj etykiety węzłów

W prawym dolnym rogu okna grafu znajdziesz pasek sterowania renderowaniem. Kliknij ikonę T (Labels), żeby włączyć wyświetlanie nazw postaci.

🔤 Ścieżka klikania:
Dolny pasek grafu → T (Show Labels)

Rozmiar etykiet proporcjonalny do węzła

Kliknij na ikonę ustawień etykiet i włącz „Size proportional to node size" — JACK i TYLER będą mieć duże napisy, peryferyjne postacie małe.

💡 Przy dużej liczbie węzłów etykiety mogą się nakładać. Użyj suwaka Label density (dolny pasek), żeby Gephi Lite automatycznie ukrywał nakładające się etykiety — pokazuje tylko te, które mieszczą się bez kolizji.
⚠️ Węzeł JACK\nNO...TYLER ma wieloliniową nazwę — artefakt danych. Jeśli zakłóca wizualizację, usuń go w zakładce Data → Nodes (kliknij wiersz → Delete node).
Faza 6 — Filtrowanie sieci

Filtruj według Degree Range — ukryj epizodyczne postacie

Twoja sieć ma wiele węzłów z Degree = 1 (jednorazowe postacie epizodyczne). Filtr pozwala skupić się na ważniejszych postaciach.

🔍 Ścieżka klikania:
Lewy panel → Filters → Topology → Degree Range

Ustaw Min = 3, żeby zobaczyć tylko postacie z minimum 3 połączeniami. Kliknij Apply filter.

💡 Filtr jest nieinwazyjny — nie usuwa węzłów, tylko je ukrywa. Możesz go wyłączyć w dowolnym momencie klikając „Remove filter".

Filtr według wagi krawędzi

Możesz też filtrować według atrybutu weight:

Filters → Attributes → Edges → weight Range

Ustaw Min = 2 — zobaczysz tylko połączenia z wagą ≥ 2, czyli faktycznie istotne interakcje.

Faza 7 — Eksploracja interaktywna

Tryb selekcji — eksploruj pojedyncze węzły

Gephi Lite pozwala interaktywnie eksplorować sieć — kliknij na postać, żeby zobaczyć jej połączenia.

🖱️ Kliknij na węzeł JACK:
Kursor na węzeł JACK → Kliknij

W prawym panelu pojawi się lista wszystkich połączeń JACKA z wagami. Możesz też Shift+Click, żeby zaznaczyć wiele węzłów naraz.

Tryb „Ego network"

Kliknij prawym przyciskiem na węzeł TYLER → wybierz „Select neighbors" (lub analogiczną opcję) — zaznaczone zostaną wszyscy sąsiedzi TYLERA, wizualnie izolując jego egonetwork.

💡 Możesz przenosić węzły klikając i przeciągając je. Przydatne do ręcznego poprawienia układu, np. odsunięcia źle wyzycjonowanego węzła.

Eksport pliku grafu

Żeby zachować graf z obliczonymi metrykami do dalszej pracy:

Workspace → Export graph file → GEXF lub GraphML

Zapisz projekt Gephi Lite

Zapisz całą konfigurację (układ, wygląd, filtry) jako projekt, żeby móc wrócić do pracy później.

💾 Ścieżka klikania:
Workspace → Save project as... → Pobierz .json

Projekt zapisany jako .json można wczytać z powrotem przez „Open a project" — zachowuje układ, coloring, rozmiary, wszystko.

☁️ GitHub Integration: Gephi Lite pozwala zapisać projekt bezpośrednio do GitHub Gist. Kliknij Workspace → Connect to GitHub, autoryzuj aplikację i możesz synchronizować pracę między urządzeniami — przydatne do udostępniania wizualizacji.
Bonus — Zaawansowane funkcje

Custom Scripts

Workspace → Custom scripts. Możesz pisać JavaScript, który modyfikuje atrybuty węzłów. Np. skrypt automatycznie nadający kolor według progu degree.

Share as URL

Workspace → Share graph as URL. Gephi Lite może zakodować cały stan wizualizacji w URL i wygenerować link do udostępnienia — bez serwera.

Attribute Model

W Data → Nodes możesz edytować typ atrybutu weight z Qualitative na Quantitative — to odblokuje pełne możliwości filtrowania i rankingu wagowego.

Layout Quality

Layout → Layout Quality → Connected-closeness. Eksperymentalna siatka pokazuje jak dobrze algorytm zgrupował połączone węzły. Sprawdź po ForceAtlas 2.

Kompletna kolejność kroków

  1. Import graf_postaci.graphml
  2. Przeglądnij dane (Data tab)
  3. ForceAtlas 2 (Scaling 100, Prevent Overlap)
  4. Metryki: Degree, Weighted Degree, Modularity, Betweenness
  5. Rozmiar węzłów → Degree (5–40)
  6. Kolor węzłów → Modularity Class
  7. Grubość krawędzi → weight (0.5–8)
  8. Etykiety + size proportional
  9. Filtr: Degree ≥ 3 lub weight ≥ 2
  10. Eksploracja interaktywna (klik na węzeł)
  11. Export PNG 4K
  12. Save project .json
